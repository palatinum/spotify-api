#  Inicializar el proyecto

## 1. Clonar el proyecto
   ```bash
   git clone https://github.com/palatinum/spotify-api.git
   ```

## 2. Crear el .env
   ```bash
   cd spotify-api/ && cp .env.example .env
   ```

## 3. Ejecutar docker:
   ```bash
   docker compose up
   ```
